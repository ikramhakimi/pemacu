# UI Copywriting

Write clean, minimal, and functional UI copy that fits inside structured layouts.

## Tone & Style
- Clear and concise
- Neutral, calm, modern
- No fluff, no hype
- Slightly premium, but not salesy

## Rules
- Keep text short and scannable
- Avoid long sentences
- Avoid generic phrases:
  - "We provide solutions"
  - "Best in class"
  - "Cutting-edge"

## UI Constraints
- Must fit inside components (buttons, cards, headers)
- Prefer 1–6 words for labels
- Max 1–2 lines for descriptions

## Common Patterns

### Labels
- "Email Address"
- "Password"
- "Search Projects"

### Buttons
- "Create Project"
- "Save Changes"
- "Invite Team"

Avoid:
- "Click Here"
- "Submit Now"

### Section Titles
Avoid:
- "Our Services"
- "What We Do"

Prefer:
- Outcome-driven titles
- Specific wording

---

## Output Rules
- Return copy only
- No explanation
- No lorem ipsum

## Self-check
- Is this too long?
- Is this generic?
- Does it fit UI space?
- Is it clear instantly?
